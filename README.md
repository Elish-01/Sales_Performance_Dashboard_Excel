# Sales_Performance_Dashboard_Excel
Sales performance analysis using Excel and interactive dashboards with insights.
## Project Overview

This project analyzes sales data to uncover key business insights such as top performing products, regions and salesperson. Power query editor made cleaning the raw data easy and quick, and an interactive dashboard was created using Microsoft Excel to visualize trends and patterns and support decision making.

## Tools Used

* Microsoft Excel
* Power Query Editor
* Pivot Table
* Data Visualization
* Slicer

## Dataset

The Dataset Includes
* Data
* Region
* Products
* Salesperson
* Units Sold
* Revenue

## Dashboard Features

* Interacitve Filters using Slicers
* Revenue analysis by Products, Regions and Salesperson
* Adding KPI Cards For Immediate Insights
* Visual Representation using charts
* Time Series analysis ( to know the pattern of change in sales performance over a periof of time (here over months))

## Key Insights

* Headphones generated the highest revenue
* North region recorded the highest sales.
* Elish is the top-performing salesperson.
* Revenue showed fluctuations throughout the year( Time Series Analysis), peaks in May and August and significant drop in September, radiating incosistent sales performance rather than a steady growth.

## Leaning Takeways
* Data cleaning and structuring
* Pivot table and charts for quick analysis
* Creating interactive dashboards along with slicers
* Extracting meaningful insigts from raw data
